==========
This assignment demonstrates the use of:
- Inline CSS (index.html): Background color for the profile card.
- Internal CSS (index.html): Styles defined within <style> tag for h1, p, and .contact p.
- External CSS (style.css): Styles for layout (.profile-card), images (#profile-pic), hover effects (a:hover), and global settings (*).

Selectors used:
- Element selector: p
- Class selector: .bio, .profile-card
- ID selector: #name, #profile-pic
- Grouping selector: h1, h2
- Descendant selector: .contact p
- Attribute selector: a[href^="mailto"], a[href^="https"]
